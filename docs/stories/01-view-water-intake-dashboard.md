# User Story: 1 - View Water Intake Dashboard

**As a** user,
**I want** to see a clear water intake counter on the app's landing page,
**so that** I can immediately see my hydration progress for the day at a glance.

## Acceptance Criteria

*   The landing page displays a prominent water intake dashboard as the primary element.
*   The dashboard shows the number of glasses consumed out of the daily goal (e.g., "3 / 8 glasses").
*   A visual indicator (e.g., progress bar, filled glass icons) communicates progress intuitively.
*   The dashboard reflects the current day's intake and resets at the start of each new day.
*   The user can log a glass of water directly from the dashboard with a single action.

## Notes

*   This is the **priority feature** and the first story to be delivered, as agreed in the meeting.
*   The default daily goal is 8 glasses; see Story 2 for goal customization.
