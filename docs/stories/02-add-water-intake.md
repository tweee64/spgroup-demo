# User Story: 02 - Add Water Intake

**As a** user,
**I want** to log when I drink a glass of water,
**so that** I can track my progress toward my daily hydration goal.

## Acceptance Criteria

* Users can easily add a glass of water to their daily count with a simple action (button/tap)
* The water counter increments immediately when water is logged
* Visual feedback confirms the action was successful
* Users can add multiple glasses throughout the day
* The interface prevents adding negative amounts of water
* Water intake logged is associated with the current date

## Notes

* This is a core interaction that should be quick and easy
* Consider making the action prominent and accessible
* Should work seamlessly on both desktop and mobile devices
