# User Story: 03 - Customizable Daily Water Goal

**As a** user,
**I want** to customize my daily water intake goal,
**so that** I can set a target that matches my personal hydration needs.

## Acceptance Criteria

* User can change the daily water goal from the default 8 glasses
* Custom goal persists across sessions
* Dashboard reflects the custom goal in progress calculations
* User can easily update their goal at any time
* Valid input range is enforced (e.g., 1-20 glasses)

## Notes

* Default is 8 glasses per day as mentioned in the transcript
* Goal should be saved to local storage or user preferences
* Consider adding recommended goals based on user profile in future
