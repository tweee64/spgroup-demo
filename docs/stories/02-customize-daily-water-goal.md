# User Story: 2 - Customize Daily Water Goal

**As a** user,
**I want** to customize my daily water intake goal,
**so that** my hydration target matches my personal needs rather than a generic default.

## Acceptance Criteria

*   The app provides a default goal of 8 glasses per day.
*   The user can navigate to a settings or profile area to change their daily water goal.
*   The updated goal is immediately reflected on the water intake dashboard.
*   The custom goal persists across sessions (i.e., it is saved and not reset on app restart).
*   The input validates that the goal is a positive, reasonable number.

## Notes

*   The PO explicitly stated: "8 glasses per day is the default, but users should be able to customize."
