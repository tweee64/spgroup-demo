# User Story: 04 - Contact Property Owner

**As a** prospective tenant,
**I want** to directly message property owners within the app,
**so that** I can ask questions and communicate seamlessly without leaving the platform.

## Acceptance Criteria

* Contact functionality is available on listing detail pages
* Users can initiate a conversation with property owners
* Messaging interface includes:
  - Real-time chat functionality
  - Message history
  - Typing indicators
  - Read receipts
* Notifications for new messages
* Message thread is associated with specific listing
* Users can attach questions or request additional information
* Both users and landlords can access message threads from their dashboard
* Basic moderation to prevent spam or inappropriate messages

## Notes

* Real-time messaging requires WebSocket or similar technology
* For MVP, consider a simpler contact form as alternative if chat is complex
* Notification system needed for new messages (email and/or in-app)
* Mobile notifications important for timely responses
