# User Story: 05 - Daily Activity Streaks and Badges

**As a** user,
**I want** to see my activity streaks and earn badges for milestones,
**so that** I stay motivated to maintain healthy daily routines.

## Acceptance Criteria

* Dashboard shows current streak count for consecutive days meeting goals
* Visual streak indicator is prominent and encouraging
* Badges are awarded for significant milestones (e.g., 7-day, 30-day streaks)
* Badge collection is displayed in user profile or achievements section
* Streak resets appropriately when goal is not met
* Celebratory feedback when badges are earned

## Notes

* Mentioned by designer as motivation feature
* Should work across all tracking types (water, sleep, steps, meditation)
* Consider gamification elements to encourage consistent use
* Future enhancement: social sharing of achievements
