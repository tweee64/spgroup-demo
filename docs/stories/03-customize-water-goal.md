# User Story: 03 - Customize Water Goal

**As a** user,
**I want** to customize my daily water intake goal,
**so that** I can set a target that matches my personal hydration needs.

## Acceptance Criteria

* Users can access a settings or configuration option to change their daily water goal
* Default goal starts at 8 glasses per day
* Users can set any reasonable positive number of glasses as their goal (e.g., 1-20 glasses)
* The dashboard updates to reflect the new goal immediately
* Custom goal persists across browser sessions
* Progress percentage recalculates based on the new goal

## Notes

* Different people have different hydration needs based on activity level, climate, etc.
* Goal should be validated to prevent unreasonable values (e.g., 0 or negative numbers)
* Consider whether to allow fractional glass values or keep it to whole numbers
