# User Story: 03 - View Listings on Map

**As a** user unfamiliar with the area,
**I want** to view available accommodations on an interactive map,
**so that** I can visually understand the locations and find housing in my preferred area.

## Acceptance Criteria

* Map view is available alongside list view (toggle or split screen)
* Map displays markers for all listings matching current search filters
* Map markers show:
  - Listing price
  - Property type icon/indicator
* Clicking on a map marker shows:
  - Preview card with photo, price, and basic details
  - Link to full listing details
* Map supports:
  - Zoom in/out functionality
  - Pan/drag to explore different areas
  - Cluster markers when many listings are close together
* Map view and list view stay synchronized with applied filters
* Map integrates with a reliable maps API (e.g., Google Maps, Mapbox)
* Mobile-responsive map interface

## Notes

* Especially useful for people relocating or unfamiliar with the area
* Real-time availability should be reflected on the map
* Consider showing nearby amenities (transit, schools, etc.) as future enhancement
