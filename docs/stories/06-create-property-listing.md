# User Story: 06 - Create Property Listing

**As a** property owner or landlord,
**I want** to create a detailed listing for my accommodation,
**so that** I can attract potential tenants and showcase my property effectively.

## Acceptance Criteria

* Listing creation form includes:
  - Property title
  - Detailed description
  - Location (address with map picker)
  - Price per month
  - Room type (studio, shared apartment, etc.)
  - Available facilities (checkboxes for Wi-Fi, gym, laundry, AC, etc.)
  - Lease duration options
  - Availability calendar
  - Photo upload (minimum 5 photos required)
* Photo upload features:
  - Multiple image upload
  - Drag and drop interface
  - Image preview before upload
  - Ability to reorder images
  - Guidelines for image quality displayed
* Form validation ensures:
  - All required fields completed
  - Minimum number of photos uploaded
  - Valid price and location data
* Draft saving capability
* Preview of listing before publishing
* Upon publishing, listing appears in search results

## Notes

* Image quality guidelines needed to ensure consistency
* Consider photo categories (bedroom, bathroom, common areas, exterior)
* Support for landlord verification could increase trust
* Ability to edit listing after creation needed
