# User Story: 01 - Search Accommodations with Filters

**As a** student or working adult looking for accommodation,
**I want** to search for housing with multiple filters (location, price, room type, lease duration, facilities),
**so that** I can quickly find suitable long-term accommodation that meets my specific needs.

## Acceptance Criteria

* Search interface is prominently displayed on the main page
* Users can filter by:
  - Location (with autocomplete or dropdown)
  - Price range (with min/max sliders or inputs)
  - Room type (studio, shared apartment, student housing, co-living spaces)
  - Lease duration (minimum stay requirements)
  - Facilities (Wi-Fi, gym, laundry, air conditioning, etc.)
* Multiple filters can be selected simultaneously
* Search results update dynamically as filters are applied
* Results display in both list and map views
* Each result shows key information: location, price, room type, and preview image
* Clear indication of active filters with ability to remove individual filters
* Efficient backend filtering for quick response times

## Notes

* This is a core MVP feature - highest priority
* Backend needs structured data for each listing to support efficient filtering
* Consider saving filter preferences for returning users (future enhancement)
* Mobile-friendly interface is essential as many users will search on phones
